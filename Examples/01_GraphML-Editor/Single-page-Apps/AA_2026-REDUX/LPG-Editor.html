LPG Graph Editor

A browser-based Labeled Property Graph editor — no server, no install.

What is this?

LPG Editor lets you visually build and explore Labeled Property Graphs — nodes and edges, each with arbitrary key/value properties, labels, and styling. Everything is stored in your browser's localStorage and can be exported as JSON, GraphML, SVG, or PNG.

⚠️ No data is sent anywhere. Your graph lives only in this tab. Export regularly to avoid losing work on refresh or browser clear.

Property Editor

Every node and edge can hold unlimited custom key/value properties. Access them three ways: (1) the Custom Properties section in the inspector panel, (2) the ⤢ expand button to open the full-screen modal editor with resizable text areas for long values, or (3) double-click any node body or edge to jump straight to the modal.

Features

Node InspectorClick any node: edit label, ID, color, radius, label font size, and custom properties.
Edge InspectorClick any edge: edit label, source/target (with autocomplete), direction, style, dash, color, and custom properties.
Merge Graphs➕ OBJs in toolbar: paste a second JSON graph and resolve ID conflicts with keep/replace per object.
Auto LayoutForce-directed layout. Positions are saved and survive refresh.
SVG / PNG ExportExport the canvas as vector SVG or raster PNG with the active theme.
GraphML ExportStandard XML readable by Gephi, yEd, Neo4j, and other tools.
Collapsible PanelsInspector (◀ button), object table, and GraphML drawer can all be hidden to maximize canvas space.
Hover TooltipHover any node to see its label and all custom properties in a floating card.

Data & Persistence

Graph state (nodes, edges, positions, colors, view zoom) auto-saves to localStorage on every meaningful action. This survives page reload within the same browser, but is wiped if you clear browser storage. A reminder banner appears after 90 seconds of unexported changes. Use Export JSON for a permanent backup.
